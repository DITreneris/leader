# Screenshot Assets

Use this folder for optimized product mockups and demo screenshots.

Recommended formats:

- AVIF first for screenshots and larger illustrations.
- WebP fallback where needed.
- PNG only when transparency or source meme quality requires it.

Suggested names:

- `demo-clarity-check.avif`
- `hero-executive-output.avif`
- `before-after-workflow.avif`
